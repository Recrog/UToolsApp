Gite Klasör Yükleme Rehberi

Cursor'da açık olan klasörünüzden Gite'a tüm klasörü yükleme adımları

Adım Adım Gite Klasör Yükleme

1

Gite Hesabınıza Giriş Yapın

Gite hesabınıza giriş yapın ve yeni bir depo (repository) oluşturun veya var olan bir depoyu açın.

2

Cursor'da Terminali Açın

Cursor'da açık olan klasörünüzde terminali açın (Terminal > New Terminal veya Ctrl+Shift+`).

3

Git'i Başlatın

Aşağıdaki komutu terminale yazarak Git'i başlatın:

git init
4

Tüm Klasörü Ekleyin

Tüm klasörü staging area'ya eklemek için:

git add .

Not: "." tüm dosya ve klasörleri ekler. Belirli dosyaları hariç tutmak için .gitignore dosyası kullanabilirsiniz.

5

Commit Oluşturun

Değişiklikleri commit edin:

git commit -m "İlk commit"
6

Ana Dalı Oluşturun (Gerekirse)

git branch -M main
7

Uzak Depoyu Ekleyin

Gite'daki deponuzun URL'sini ekleyin:

git remote add origin https://gite.com/kullanici_adi/depo_adi.git

Not: URL'yi kendi Gite deponuzun URL'si ile değiştirin.

8

Klasörü Gönderin

Tüm klasörü Gite'a gönderin:

git push -u origin main
9

Kimlik Doğrulama

İstendiğinde Gite kullanıcı adı ve şifrenizi girin veya erişim token'ınızı kullanın.

10

Doğrulama

Gite'daki deponuzu kontrol ederek klasörün başarıyla yüklendiğini doğrulayın.

Hızlı Komut Listesi

  • git init - Git deposu başlatma
  • git status - Durumu kontrol etme
  • git add . - Tüm dosya ve klasörleri ekleme
  • git commit -m "mesaj" - Commit oluşturma
  • git branch -M main - Ana dalı oluşturma
  • git remote add origin [url] - Uzak depo ekleme
  • git push -u origin main - Klasörü gönderme
  • git clone [url] - Depoyu klonlama
